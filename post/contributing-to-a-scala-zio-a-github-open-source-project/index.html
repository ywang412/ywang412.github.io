Contributing to Scala zio, a github open source project

· 2019-02-19 · # Scala

I've hoped to contribute to open source project on github to be a part of the open source community and become a better developer. Today my two pull requests to a high profile scala repository, zio, got approved and I am very excited. The friendly and encouraging github community brought me a great experience.

These are the two pull requests that I made.

https://github.com/zio/zio/pull/1109
and
https://github.com/zio/zio/pull/1106

The first one is to deprecate a FiberLocal class.

The second one is to fix a bug resulting from different Windows/Linux file seperator.

The zio scala repository is very well maintained with decent test coverage and CD/CI pipeline. Thanks to the project maintainers for their huge efforts.
