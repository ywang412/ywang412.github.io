[Scala] Flat a arbitrarily-deep nested list

2019-06-25

It is easy to figure out a recursive solution which tries to flat each element in a nested list. The problem is how to aggregate the elements into a new list. Foldleft() with an initial empty list will work but a more elegant solution is to create an one element list if the element is an integer. Below is the comparison.

Foldleft

object FlattenArray {
  def flatten(list : List[Any]) : List[Int] = {
    list.foldLeft(List[Int]()){
      (res, cur) => {
        cur match {
          case null => res
          case cur: Int => res :+ cur
          case cur: List[Any] => res ::: flatten(cur)
        }
      }
    }
  }
}

Create single element list

object FlattenArray {
  def flatten(input: List[_]): List[_] =
    input
      .filterNot(_ == null)
      .flatMap {
        case list: List[_] => flatten(list)
        case elem => List(elem)
      }
}
